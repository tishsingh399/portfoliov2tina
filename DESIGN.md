# Portfolio Design System Contract

## Visual Theme & Atmosphere
- Premium, professional portfolio layout.

## Component Stylings
- **Images**: All images MUST use local paths from the `/images` directory (e.g., `images/file.png`). DO NOT use external image URLs or upload to Imgur.
- **Image Dimensions**: All images must have a fixed width of `600px` and `object-fit: cover`. Do not use 100% widths that cause layout issues.

## Do's and Don'ts
- **DO NOT** use AI-generated placeholders or placeholders from external domains.
- **DO** refer to this `DESIGN.md` file before making any CSS or HTML changes.
- **DO** assume local images will be placed in the `/images` folder by the user.
